План навчальної практики з ООП: Каталог рецептів



## Опис проєкту

Веб-додаток для збереження та перегляду рецептів з можливістю автоматичного перерахунку інгредієнтів при зміні кількості порцій.



## Основні функції

- Додавання рецептів

- Збереження рецептів у базі даних

- Перерахунок інгредієнтів

- Парсер рецептів за посиланням

- Каталог рецептів


## Технології

- Frontend: React / Vue

- Backend: Node.js / Python

- База даних: SQLite / MongoDB

- Python Parser: BeautifulSoup

## Запуск проекту

1. Встановіть Python 3.10+ та активуйте віртуальне середовище.
2. Запустіть сервер із кореневої директорії проєкту:

```powershell
python app.py
```

3. Відкрийте у браузері:

```text
http://127.0.0.1:8000
```

Сторінка завантажує рецепти з `global_recipes_database.json` через API `/api/recipes`.
