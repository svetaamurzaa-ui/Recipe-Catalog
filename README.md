План навчальної практики з ООП: Каталог рецептів



## Опис проєкту

Веб-додаток для збереження та перегляду рецептів з можливістю автоматичного перерахунку інгредієнтів при зміні кількості порцій.



## Основні функції

- Додавання рецептів

- Збереження рецептів у базі даних

- Перерахунок інгредієнтів

- Парсер рецептів за посиланням

- Каталог рецептів


## Технології

- Frontend: React / Vue

- Backend: Node.js / Python

- База даних: SQLite / MongoDB

- Python Parser: BeautifulSoup
